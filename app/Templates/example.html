While using this website to predict, you have two options:
1) You can enter a file with one or multiple row of data, each corresponding to a woman.
2) You can enter the data manually for one woman only. To switch between the two options, you press the corresponding button in the top left.

If you choose to use the file, you should click the top option, and choose the file from your computer, it should look like this:

Meanwhile, if you choose to enter manually, you either enter yes or no, or the correct amount for each numerical box. All boxes must be filled out. After you fill it out, it should look like this:

Now, you should click the "Calculate" button, and it will calculate the probability that the patient has GDM. If you entered multiple rows in the file, the website should look like the top plot, and if you entered the data manually or with a single row, it should look like the bottom.
